# README

## INSTALL

  - git clone `https://github.com/fauzipadlaw/spk-topsis.git`
  - `cd spk-topsis && bundle install`
  - configure database in `config/database.yml`
  - run `rails db:create`
  - run `rails db:migrate`
  - run `rails db:seed`
  - run `rails s`
  - Open `localhost:3000` on your browser
